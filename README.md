# yolov5-flask-video-stream
A simple implementation for live streaming object detection with YOLOv5 and Flask.

## Install
```bash
pip install -r requirements.txt
```
## Useage
```bash
flask run --port 5000
```
For ip camera streaming: change source to a rtsp link.
